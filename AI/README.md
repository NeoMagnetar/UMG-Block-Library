# AI

This side is for machine-readable or machine-oriented UMG surfaces:
- doctrine
- schemas
- manifests
- baseline block libraries
- selected structural artifacts

This first pass is intentionally curated and partial.

## Current lane notes
- `AI/MOLT-BLOCKS/` is the current canonical machine-readable block library lane.
- `AI/BLOCKS/` is not a canonical lane in the current repo state and remains deferred unless explicitly introduced later.
- `AI/RUNTIME-REFERENCE/` is a public runtime/reference lane and remains distinct from any future deterministic `AI/FIXTURES/` lane.
- `AI/GATES/` and `AI/COMPILER/` are valid special-purpose AI lanes.
