# AI MOLT-BLOCKS

## What this section is
This section is the machine-readable MOLT block shelf. It contains canonical machine-facing libraries of small composable idea units: purpose, constraint, instruction, subject, need, use, blueprint, philosophy, and related block families.

## What this section is not
This is not the main human browsing shelf. These JSON libraries are not written first for casual reading, and they should not be confused with the readable pages under `HUMAN/MOLT-BLOCKS/`.
