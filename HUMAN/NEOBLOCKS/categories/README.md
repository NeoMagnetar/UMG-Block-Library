# HUMAN NEOBLOCK CATEGORIES

## What this section is
This section is the human-readable category bridge for NeoBlocks. It helps readers map sleeve categories and stack categories to the machine-facing NeoBlock artifacts currently stored under `AI/NEOBLOCKS/categories/`.
