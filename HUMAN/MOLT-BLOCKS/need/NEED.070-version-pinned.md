# NEED.070 - Version pinned

**Type:** NEED
**Category:** INTERFACE_CONTRACT
**Group Description:** What must be true about interfaces and contracts before systems can connect.
**Status:** active

## Definition
The specific interface version in use must be locked before work that depends on it begins.

## When to use
What must be true about interfaces and contracts before systems can connect.

## Examples
- API version must be pinned in client config
- canon version must be confirmed before compilation
- library version must be locked in package manifest
