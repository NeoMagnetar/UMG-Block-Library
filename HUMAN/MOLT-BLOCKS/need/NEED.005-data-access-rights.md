# NEED.005 - Data access rights

**Type:** NEED
**Category:** DATA_PREREQUISITES
**Group Description:** What must be true about data before it can be used.
**Status:** active

## Definition
The process must have explicit permission to read the required data.

## When to use
What must be true about data before it can be used.

## Examples
- read role must be granted before query
- API key must be scoped to the data endpoint
- governance must permit block access before compilation
