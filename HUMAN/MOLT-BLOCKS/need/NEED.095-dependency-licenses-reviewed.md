# NEED.095 - Dependency licenses reviewed

**Type:** NEED
**Category:** DEPENDENCY_RESOLUTION
**Group Description:** What must be true about dependencies before a component can function.
**Status:** active

## Definition
Legal obligations of all dependencies must be understood before they are included in a product.

## When to use
What must be true about dependencies before a component can function.

## Examples
- open source licenses must be checked before distribution
- third-party block licenses must be reviewed before ClawHub publication
- SDK terms must be accepted before commercial use
