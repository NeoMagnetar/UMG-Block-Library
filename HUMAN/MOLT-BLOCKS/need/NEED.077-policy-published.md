# NEED.077 - Policy published

**Type:** NEED
**Category:** POLICY_DEFINITION
**Group Description:** What must be true about policies and rules before they can be enforced.
**Status:** active

## Definition
The rules must be made available to all parties subject to them before enforcement.

## When to use
What must be true about policies and rules before they can be enforced.

## Examples
- governance stack must be visible to all Envoys before compilation
- data policy must be published before it is binding
- rate limit rules must be documented before API is used
