# NEED.079 - Enforcement mechanism active

**Type:** NEED
**Category:** POLICY_DEFINITION
**Group Description:** What must be true about policies and rules before they can be enforced.
**Status:** active

## Definition
A mechanism that applies the policy and detects violations must be operating before the system handles real inputs.

## When to use
What must be true about policies and rules before they can be enforced.

## Examples
- governance pass must be running before compilation
- rate limiter must be active before API is public
- access control layer must be deployed before traffic is routed
