# NEED.098 - Block library available

**Type:** NEED
**Category:** UMG_FRAMEWORK_NEEDS
**Group Description:** Prerequisites specific to UMG compilation, governance, and Envoy operation.
**Status:** active

## Definition
The blocks referenced in the sleeve must exist and be accessible before compilation proceeds.

## When to use
Prerequisites specific to UMG compilation, governance, and Envoy operation.

## Examples
- all MOLT blocks must be in the library before sleeve is compiled
- NeoBlock sources must be resolvable before merge
- governance block definitions must be accessible before governance pass
