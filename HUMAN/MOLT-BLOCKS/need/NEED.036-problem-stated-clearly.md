# NEED.036 - Problem stated clearly

**Type:** NEED
**Category:** PROBLEM_DEFINITION
**Group Description:** What must be true about a problem before diagnosis or resolution begins.
**Status:** active

## Definition
The issue must be described with enough precision to guide investigation.

## When to use
What must be true about a problem before diagnosis or resolution begins.

## Examples
- error message must be available before debugging
- governance violation must be named before it can be corrected
- user complaint must be understood before a fix is designed
