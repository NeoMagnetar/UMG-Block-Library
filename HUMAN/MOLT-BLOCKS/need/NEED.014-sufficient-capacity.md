# NEED.014 - Sufficient capacity

**Type:** NEED
**Category:** SYSTEM_PREREQUISITES
**Group Description:** What must be true about infrastructure and environment before a system can operate.
**Status:** active

## Definition
Enough compute, storage, memory, or bandwidth must be available before the task is started.

## When to use
What must be true about infrastructure and environment before a system can operate.

## Examples
- disk space must be available before write
- memory must be below threshold before heavy job
- queue must have capacity before message send
