# NEED.038 - Symptoms documented

**Type:** NEED
**Category:** PROBLEM_DEFINITION
**Group Description:** What must be true about a problem before diagnosis or resolution begins.

## Definition
Observable evidence of the problem must be recorded before root cause analysis begins.

## When to use
What must be true about a problem before diagnosis or resolution begins.

## Examples
- error logs must be captured before investigation
- trace events must be collected before audit review
- user-reported behavior must be noted before reproduction attempt
