# NEED.071 - Logging enabled

**Type:** NEED
**Category:** OBSERVABILITY_BASELINE
**Group Description:** What must be true about logging, metrics, and monitoring before a system goes live.
**Status:** active

## Definition
The system must be configured to record its actions and state changes before it handles real traffic.

## When to use
What must be true about logging, metrics, and monitoring before a system goes live.

## Examples
- application logging must be on before production deploy
- Decision Trace logging must be active before compilation
- audit log must be enabled before sensitive actions are taken
