# NEED.041 - Test environment available

**Type:** NEED
**Category:** TEST_INFRASTRUCTURE
**Group Description:** What must be true about testing environment and tooling before tests can run.
**Status:** active

## Definition
A dedicated, stable environment for running tests must exist before tests are executed.

## When to use
What must be true about testing environment and tooling before tests can run.

## Examples
- staging environment must be provisioned
- isolated test namespace must be ready
