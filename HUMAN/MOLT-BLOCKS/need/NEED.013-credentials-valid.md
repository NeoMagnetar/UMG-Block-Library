# NEED.013 - Credentials valid

**Type:** NEED
**Category:** SYSTEM_PREREQUISITES
**Group Description:** What must be true about infrastructure and environment before a system can operate.
**Status:** active

## Definition
Authentication tokens, certificates, or keys must be current and correct before access is attempted.

## When to use
What must be true about infrastructure and environment before a system can operate.

## Examples
- OAuth token must not be expired
- TLS certificate must be valid
- API key must be active and scoped correctly
