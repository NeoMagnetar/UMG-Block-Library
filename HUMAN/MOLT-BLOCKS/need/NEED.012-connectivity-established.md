# NEED.012 - Connectivity established

**Type:** NEED
**Category:** SYSTEM_PREREQUISITES
**Group Description:** What must be true about infrastructure and environment before a system can operate.
**Status:** active

## Definition
Network paths between system components must be open, tested, and reachable.

## When to use
What must be true about infrastructure and environment before a system can operate.

## Examples
- VPN must be connected before internal API access
- firewall rules must allow traffic
- service discovery must resolve before first call
