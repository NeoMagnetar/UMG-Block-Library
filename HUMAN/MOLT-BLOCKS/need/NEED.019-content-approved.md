# NEED.019 - Content approved

**Type:** NEED
**Category:** CONTENT_PREREQUISITES

## Definition
Where applicable, required approvals must be in place before content is used or published.

## When to use
What must be true about source material and context before content can be produced or processed.

## Examples
- governance pass must approve blocks before compilation
- legal must sign off before publishing
- stakeholder must review before release
