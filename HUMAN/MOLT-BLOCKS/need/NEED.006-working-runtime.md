# NEED.006 - Working runtime

**Type:** NEED
**Category:** CODE_PREREQUISITES
**Group Description:** What must be true about a codebase or runtime before execution.
**Status:** active

## Definition
A functional execution environment must be present and healthy before code can run.

## When to use
What must be true about a codebase or runtime before execution.

## Examples
- Node.js must be installed and on PATH
- Python virtual environment must be active
- JVM must be running before app starts
