# NEED.007 - Dependencies resolved

**Type:** NEED
**Category:** CODE_PREREQUISITES
**Group Description:** What must be true about a codebase or runtime before execution.
**Status:** active

## Definition
All required libraries, packages, and modules must be installed and version-compatible.

## When to use
What must be true about a codebase or runtime before execution.

## Examples
- npm install must complete without error
- pip packages must be resolved
- UMG block references must all exist in the library
