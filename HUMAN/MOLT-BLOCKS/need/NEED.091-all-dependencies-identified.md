# NEED.091 - All dependencies identified

**Type:** NEED
**Category:** DEPENDENCY_RESOLUTION
**Group Description:** What must be true about dependencies before a component can function.
**Status:** active

## Definition
Every component, service, library, or block that is required must be listed before work begins.

## When to use
What must be true about dependencies before a component can function.

## Examples
- all block references in a sleeve must be listed before compilation
- all service dependencies must be known before architecture review
- all package requirements must be declared in manifest before install
