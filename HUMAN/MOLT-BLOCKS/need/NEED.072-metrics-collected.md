# NEED.072 - Metrics collected

**Type:** NEED
**Category:** OBSERVABILITY_BASELINE
**Group Description:** What must be true about logging, metrics, and monitoring before a system goes live.
**Status:** active

## Definition
Quantitative measures of system behavior must be captured before performance can be assessed.

## When to use
What must be true about logging, metrics, and monitoring before a system goes live.

## Examples
- response time instrumentation must be in place before load test
- block compilation counts must be emitted before analysis
- error rates must be tracked before SLA is evaluated
