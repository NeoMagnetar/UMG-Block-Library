# NEED.054 - Throughput sufficient

**Type:** NEED
**Category:** COMMUNICATION_CHANNEL
**Group Description:** What must be true about channels and protocols before communication can occur.
**Status:** active

## Definition
The channel must have capacity for the required message volume before traffic is sent.

## When to use
What must be true about channels and protocols before communication can occur.

## Examples
- queue must not be full before new publish
- bandwidth must be available before file transfer
- governance channel must not be saturated before audit send
