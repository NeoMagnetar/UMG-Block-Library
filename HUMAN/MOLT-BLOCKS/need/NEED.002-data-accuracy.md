# NEED.002 - Data accuracy

**Type:** NEED
**Category:** DATA_PREREQUISITES
**Group Description:** What must be true about data before it can be used.
**Status:** active

## Definition
The data must be correct and substantially free from errors before use.

## When to use
What must be true about data before it can be used.

## Examples
- sensor readings must be calibrated
- block content must match the authored intent
- requirements must reflect what was actually agreed
