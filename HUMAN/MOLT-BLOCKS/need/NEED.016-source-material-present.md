# NEED.016 - Source material present

**Type:** NEED
**Category:** CONTENT_PREREQUISITES
**Group Description:** What must be true about source material and context before content can be produced or processed.
**Status:** active

## Definition
The content to be processed, summarized, edited, or delivered must exist before work begins.

## When to use
What must be true about source material and context before content can be produced or processed.

## Examples
- article must be written before it can be summarized
- blocks must be authored before sleeve can be compiled
- source doc must be uploaded before analysis
