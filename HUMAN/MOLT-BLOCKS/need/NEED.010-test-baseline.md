# NEED.010 - Test baseline

**Type:** NEED
**Category:** CODE_PREREQUISITES
**Group Description:** What must be true about a codebase or runtime before execution.
**Status:** active

## Definition
A known-good reference state must exist to compare results against.

## When to use
What must be true about a codebase or runtime before execution.

## Examples
- golden output must be stored before regression test
- prior compilation trace must exist for comparison
- baseline metrics must be recorded before benchmark
