# NEED.061 - Compute available

**Type:** NEED
**Category:** RESOURCE_AVAILABILITY
**Group Description:** What must be true about resources before work can proceed.
**Status:** active

## Definition
Sufficient processing capacity must be allocated for the task before it starts.

## When to use
What must be true about resources before work can proceed.

## Examples
- CPU quota must not be exhausted before job starts
- inference budget must be available before compilation
- cluster capacity must exist before deploy
