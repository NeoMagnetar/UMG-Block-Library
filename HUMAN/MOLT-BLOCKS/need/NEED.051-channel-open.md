# NEED.051 - Channel open

**Type:** NEED
**Category:** COMMUNICATION_CHANNEL
**Group Description:** What must be true about channels and protocols before communication can occur.
**Status:** active

## Definition
The communication pathway must be active and reachable before a message is sent.

## When to use
What must be true about channels and protocols before communication can occur.

## Examples
- WebSocket must be connected before event push
- SMTP server must be reachable before email send
- API endpoint must be listening before request
