# NEED.039 - Reproduction steps available

**Type:** NEED
**Category:** PROBLEM_DEFINITION
**Group Description:** What must be true about a problem before diagnosis or resolution begins.
**Status:** active

## Definition
A reliable way to trigger the problem must be known before a fix is attempted.

## When to use
What must be true about a problem before diagnosis or resolution begins.

## Examples
- steps to reproduce a bug must exist before patch
- a sleeve that fails must be available before fix
- a governance violation must be reproducible before rule is written
