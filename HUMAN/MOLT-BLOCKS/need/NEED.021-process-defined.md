# NEED.021 - Process defined

**Type:** NEED
**Category:** PROCESS_PREREQUISITES
**Group Description:** What must be true about a process or workflow before it can run.
**Status:** active

## Definition
The steps, sequence, and rules of the process must be documented before execution begins.

## When to use
What must be true about a process or workflow before it can run.

## Examples
- compiler pipeline must be defined before compilation
- workflow DAG must be authored before orchestration
- deployment runbook must exist before release
