# NEED.011 - Infrastructure online

**Type:** NEED
**Category:** SYSTEM_PREREQUISITES
**Group Description:** What must be true about infrastructure and environment before a system can operate.
**Status:** active

## Definition
The underlying servers, networks, or cloud resources must be running and reachable.

## When to use
What must be true about infrastructure and environment before a system can operate.

## Examples
- database must be up before app starts
- network must be available before API call
- cluster must be healthy before deploy
