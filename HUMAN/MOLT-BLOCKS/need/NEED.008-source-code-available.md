# NEED.008 - Source code available

**Type:** NEED
**Category:** CODE_PREREQUISITES
**Group Description:** What must be true about a codebase or runtime before execution.
**Status:** active

## Definition
The code to be executed, built, or analyzed must be accessible in the environment.

## When to use
What must be true about a codebase or runtime before execution.

## Examples
- repo must be cloned before build
- block definitions must be present before compilation
- source files must be readable before lint
