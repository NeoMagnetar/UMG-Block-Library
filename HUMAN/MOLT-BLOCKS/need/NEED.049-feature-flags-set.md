# NEED.049 - Feature flags set

**Type:** NEED
**Category:** CONFIGURATION_STATE
**Group Description:** What must be true about configuration and settings before a system can operate correctly.
**Status:** active

## Definition
The correct feature toggles must be in the expected state before the system executes.

## When to use
What must be true about configuration and settings before a system can operate correctly.

## Examples
- governance strict mode must be enabled before audit
- experimental block must be flagged OFF before production
- live compilation must be toggled on before interactive session
