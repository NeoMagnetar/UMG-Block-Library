# NEED.025 - Process tested

**Type:** NEED
**Category:** PROCESS_PREREQUISITES
**Group Description:** What must be true about a process or workflow before it can run.
**Status:** active

## Definition
The process must have been validated in a safe environment before production use.

## When to use
What must be true about a process or workflow before it can run.

## Examples
- pipeline must pass integration tests before production deploy
- sleeve must be tested before Envoy activation
- workflow must run in staging before going live
