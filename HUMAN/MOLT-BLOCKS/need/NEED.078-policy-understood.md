# NEED.078 - Policy understood

**Type:** NEED
**Category:** POLICY_DEFINITION
**Group Description:** What must be true about policies and rules before they can be enforced.

## Definition

## When to use
What must be true about policies and rules before they can be enforced.

## Examples
- Envoy must understand which blocks are OFF before composing a sleeve
- team must understand GDPR obligations before handling data
- user must know rate limits before building an integration
