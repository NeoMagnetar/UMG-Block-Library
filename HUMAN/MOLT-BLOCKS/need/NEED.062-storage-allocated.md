# NEED.062 - Storage allocated

**Type:** NEED
**Category:** RESOURCE_AVAILABILITY
**Group Description:** What must be true about resources before work can proceed.
**Status:** active

## Definition
Enough storage for inputs, outputs, and state must be reserved before the process runs.

## When to use
What must be true about resources before work can proceed.

## Examples
- disk space must be confirmed before log write
- S3 bucket must exist before artifact upload
- CIR storage must be available before compilation
