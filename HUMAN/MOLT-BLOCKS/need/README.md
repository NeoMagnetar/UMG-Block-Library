# NEED

Comprehensive library of atomic MOLT Need blocks. Need answers: 'What must be true first?' Each block describes a prerequisite, enabling condition, or required state that must exist before the Aim can be achieved.

## Family role
NEED answers: 'What must be true first?' It frames the prerequisite, constraint, or non-optional condition required for success.

## How to browse
Use `INDEX.md` for category-grouped browsing, or open individual `NEED.###-*.md` files for one block at a time.

## Relationship to other families
- NEED defines what must hold true
- AIM defines what is being pursued
- USE defines what a tool or method is for
- VERIFICATION can test whether the need was met

## Files
- `INDEX.md` - grouped block index
- `NEED-LIBRARY-v1.0.0.md` - human-facing family overview
- `AI/MOLT-BLOCKS/need.library.v1.0.0.json` - machine-facing source library
