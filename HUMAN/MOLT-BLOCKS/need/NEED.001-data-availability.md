# NEED.001 - Data availability

**Type:** NEED
**Category:** DATA_PREREQUISITES
**Group Description:** What must be true about data before it can be used.
**Status:** active

## Definition
The required data must exist and be accessible to the process before it can proceed.

## When to use
What must be true about data before it can be used.

## Examples
- logs must be present before analysis
- user records must exist before profiling
- block library must be loaded before compilation
