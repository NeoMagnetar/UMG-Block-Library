# NEED.085 - Timeliness confirmed

**Type:** NEED
**Category:** DATA_QUALITY_BASELINE
**Group Description:** What must be true about data quality before analysis or decisions can rely on it.
**Status:** active

## Definition
Data must fall within the acceptable age window for the use case before it is used.

## When to use
What must be true about data quality before analysis or decisions can rely on it.

## Examples
- block library must be current canon version before compilation
- market data must be from today before trading decision
- security scan results must be recent before deployment approval
