# NEED.022 - Roles assigned

**Type:** NEED
**Category:** PROCESS_PREREQUISITES
**Group Description:** What must be true about a process or workflow before it can run.
**Status:** active

## Definition
Each step in a process must have a designated owner or executor before the process starts.

## When to use
What must be true about a process or workflow before it can run.

## Examples
- each governance block must have an author
- each stage must have an assigned reviewer
- each alert must have a responsible team
