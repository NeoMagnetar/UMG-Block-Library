# NEED.076 - Policy authored

**Type:** NEED
**Category:** POLICY_DEFINITION
**Group Description:** What must be true about policies and rules before they can be enforced.
**Status:** active

## Definition
The governing rules must be written in explicit, inspectable form before enforcement begins.

## When to use
What must be true about policies and rules before they can be enforced.

## Examples
- governance block must be authored before governance pass runs
- data retention policy must be written before it is enforced
- access control rules must be declared before permission checks
