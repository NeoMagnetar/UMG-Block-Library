# NEED.050 - Configuration versioned

**Type:** NEED
**Category:** CONFIGURATION_STATE
**Group Description:** What must be true about configuration and settings before a system can operate correctly.
**Status:** active

## Definition
The configuration in use must be traceable to a specific documented version before deployment.

## When to use
What must be true about configuration and settings before a system can operate correctly.

## Examples
- config must be committed to source control before deploy
- canon version must be confirmed before compilation
- infrastructure manifest must be tagged before apply
