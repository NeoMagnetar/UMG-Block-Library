# NEED.047 - Environment variables set

**Type:** NEED
**Category:** CONFIGURATION_STATE
**Group Description:** What must be true about configuration and settings before a system can operate correctly.
**Status:** active

## Definition
Required runtime variables must be present and correct in the environment before execution.

## When to use
What must be true about configuration and settings before a system can operate correctly.

## Examples
- DATABASE_URL must be set before app start
- GOVERNANCE_MODE must be declared before compilation
- API_KEY must be injected before first call
