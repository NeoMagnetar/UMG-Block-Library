# NEED.092 - Versions compatible

**Type:** NEED
**Category:** DEPENDENCY_RESOLUTION
**Group Description:** What must be true about dependencies before a component can function.
**Status:** active

## Definition
All dependency versions must be mutually compatible before they are used together.

## When to use
What must be true about dependencies before a component can function.

## Examples
- block library version must be compatible with compiler version
- package versions must not have peer conflicts
- canon version must be agreed across all Envoys before mesh sync
