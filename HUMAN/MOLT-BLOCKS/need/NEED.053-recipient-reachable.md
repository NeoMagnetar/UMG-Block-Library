# NEED.053 - Recipient reachable

**Type:** NEED
**Category:** COMMUNICATION_CHANNEL
**Group Description:** What must be true about channels and protocols before communication can occur.
**Status:** active

## Definition
The target of the message must be available to receive before sending.

## When to use
What must be true about channels and protocols before communication can occur.

## Examples
- service must be healthy before routing to it
- user must be active before push notification
- Envoy must be initialized before receiving a MOLT block
