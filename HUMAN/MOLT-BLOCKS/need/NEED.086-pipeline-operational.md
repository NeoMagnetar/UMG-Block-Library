# NEED.086 - Pipeline operational

**Type:** NEED
**Category:** DELIVERY_READINESS
**Group Description:** What must be true about delivery pipelines and processes before releasing.
**Status:** active

## Definition
The deployment or delivery mechanism must be running and tested before it carries production traffic.

## When to use
What must be true about delivery pipelines and processes before releasing.

## Examples
- CI/CD pipeline must be passing before merge to main
- compilation pipeline must complete without error before RuntimeSpec is trusted
- delivery channel must be tested before release
