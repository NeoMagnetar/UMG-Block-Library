# NEED.100 - Compiler initialized

**Type:** NEED
**Category:** UMG_FRAMEWORK_NEEDS
**Group Description:** Prerequisites specific to UMG compilation, governance, and Envoy operation.
**Status:** active

## Definition
The UMG compiler must be ready, with canon version confirmed, before a compilation pass is started.

## When to use
Prerequisites specific to UMG compilation, governance, and Envoy operation.

## Examples
- compiler canon version must match sleeve spec version before compile
- compiler must be initialized before sleeve ingestion
- compiler state must be clean before each compilation session
