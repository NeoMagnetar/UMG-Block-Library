# NEED.030 - Scope bounded

**Type:** NEED
**Category:** ACCESS_AUTHORIZATION
**Group Description:** What must be true about identity and permission before action is permitted.
**Status:** active

## Definition
Access must be explicitly limited to the minimum required for the task.

## When to use
What must be true about identity and permission before action is permitted.

## Examples
- API token must be scoped to read-only
- governance block must limit tool access before execution
- service account must not have broader rights than needed
