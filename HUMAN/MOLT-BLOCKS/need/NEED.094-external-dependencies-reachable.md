# NEED.094 - External dependencies reachable

**Type:** NEED
**Category:** DEPENDENCY_RESOLUTION
**Group Description:** What must be true about dependencies before a component can function.
**Status:** active

## Definition
Third-party services, APIs, or libraries must be available before the system starts.

## When to use
What must be true about dependencies before a component can function.

## Examples
- npm registry must be reachable before install
- external API must be up before integration test
- external block library must be accessible before compilation
