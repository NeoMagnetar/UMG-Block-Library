# NEED.066 - API contract defined

**Type:** NEED
**Category:** INTERFACE_CONTRACT
**Group Description:** What must be true about interfaces and contracts before systems can connect.
**Status:** active

## Definition
The interface specification must be agreed and published before any client or server implements it.

## When to use
What must be true about interfaces and contracts before systems can connect.

## Examples
- OpenAPI spec must be finalized before SDK is built
- RuntimeSpec schema must be stable before executor is coded
- MOLT block interface must be defined before first authoring tool
