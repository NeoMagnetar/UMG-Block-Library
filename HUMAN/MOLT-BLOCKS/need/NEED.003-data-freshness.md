# NEED.003 - Data freshness

**Type:** NEED
**Category:** DATA_PREREQUISITES
**Group Description:** What must be true about data before it can be used.
**Status:** active

## Definition
The data must be recent enough to reflect current state for the decision at hand.

## When to use
What must be true about data before it can be used.

## Examples
- metrics must be from the last 24 hours
- block library must be current canon
- inventory data must be updated before stock check
