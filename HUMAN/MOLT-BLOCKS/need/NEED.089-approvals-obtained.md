# NEED.089 - Approvals obtained

**Type:** NEED
**Category:** DELIVERY_READINESS

## Definition

## When to use

## Examples
- security review must be signed off before production deploy
- governance owner must approve rule change before enforcement
- product must sign off on feature before ship
