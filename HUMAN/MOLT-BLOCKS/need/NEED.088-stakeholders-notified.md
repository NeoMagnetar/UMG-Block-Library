# NEED.088 - Stakeholders notified

**Type:** NEED
**Category:** DELIVERY_READINESS
**Group Description:** What must be true about delivery pipelines and processes before releasing.
**Status:** active

## Definition
Those affected by the delivery must be informed in advance before it proceeds.

## When to use
What must be true about delivery pipelines and processes before releasing.

## Examples
- users must be notified of maintenance window before deploy
- downstream Envoys must be informed of canon change before it is released
- affected teams must be briefed before a breaking API change
