# NEED.028 - Permissions granted

**Type:** NEED
**Category:** ACCESS_AUTHORIZATION
**Group Description:** What must be true about identity and permission before action is permitted.
**Status:** active

## Definition
Explicit permission for the requested action must exist in the access control system.

## When to use
What must be true about identity and permission before action is permitted.

## Examples
- write permission must exist before file is modified
- governance must permit the block before it is compiled
- user must have delete right before resource removal
