# NEED.024 - Exception handling defined

**Type:** NEED
**Category:** PROCESS_PREREQUISITES
**Group Description:** What must be true about a process or workflow before it can run.
**Status:** active

## Definition
Failure paths and edge cases must have documented responses before the process goes live.

## When to use
What must be true about a process or workflow before it can run.

## Examples
- error recovery block must be in place before pipeline runs
- rollback procedure must exist before deployment
- governance fail-fast behavior must be defined before compilation
