# NEED.023 - Triggers configured

**Type:** NEED
**Category:** PROCESS_PREREQUISITES
**Group Description:** What must be true about a process or workflow before it can run.
**Status:** active

## Definition
The conditions that start or advance the process must be set and tested before the process runs.

## When to use
What must be true about a process or workflow before it can run.

## Examples
- CI trigger must be configured before first push
- UMG trigger blocks must be authored before evaluation
- scheduled job must be registered before first run
