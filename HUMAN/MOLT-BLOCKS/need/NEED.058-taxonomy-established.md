# NEED.058 - Taxonomy established

**Type:** NEED
**Category:** STRUCTURAL_FOUNDATION
**Group Description:** What must be true about models, schemas, and structures before reasoning can proceed.
**Status:** active

## Definition
Categories and classifications must be agreed and documented before things are organized into them.

## When to use
What must be true about models, schemas, and structures before reasoning can proceed.

## Examples
- MOLT type hierarchy must be defined before blocks are typed
- permission levels must be named before roles are assigned
- product categories must exist before items are cataloged
