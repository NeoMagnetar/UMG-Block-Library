# AIM.022 - Restore normal operation

**Type:** AIM
**Category:** RESOLUTION_OUTCOMES
**Group Description:** Aims oriented toward fixing, repairing, and preventing failures.
**Status:** active

## Definition
To return a disrupted system or process to its expected state.

## When to use
Aims oriented toward fixing, repairing, and preventing failures.

## Examples
- recovering a downed service
- re-enabling an OFF'd block
- resuming a failed pipeline
