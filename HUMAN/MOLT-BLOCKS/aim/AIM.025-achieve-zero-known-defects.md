# AIM.025 - Achieve zero known defects

**Type:** AIM
**Category:** RESOLUTION_OUTCOMES
**Group Description:** Aims oriented toward fixing, repairing, and preventing failures.
**Status:** active

## Definition
To reach a state where no identified failures remain unresolved.

## When to use
Aims oriented toward fixing, repairing, and preventing failures.

## Examples
- clearing the bug backlog for release
- resolving all open ODRs
- closing all failing tests before ship
