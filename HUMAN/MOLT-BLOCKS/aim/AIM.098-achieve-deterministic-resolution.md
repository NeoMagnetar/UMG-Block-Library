# AIM.098 - Achieve deterministic resolution

**Type:** AIM
**Category:** UMG_FRAMEWORK_AIMS
**Group Description:** Aims specific to achieving correct, governed, auditable UMG operation.
**Status:** active

## Definition
To ensure identical inputs always produce identical compiled outputs.

## When to use
Aims specific to achieving correct, governed, auditable UMG operation.

## Examples
- confirming two compiler runs on the same sleeve produce the same RuntimeSpec
- verifying no random or context-sensitive resolution occurs
- proving compilation is reproducible across Envoy instances
