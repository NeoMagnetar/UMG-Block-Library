# AIM.075 - Achieve end-to-end traceability

**Type:** AIM
**Category:** INTEGRATION_OUTCOMES
**Group Description:** Aims oriented toward connecting, unifying, and achieving coherence.
**Status:** active

## Definition
To track a transaction, decision, or artifact across every system it passes through.

## When to use
Aims oriented toward connecting, unifying, and achieving coherence.

## Examples
- tracing a block from authoring to RuntimeSpec
- following a request from entry to response
- linking a requirement to a deployed feature
