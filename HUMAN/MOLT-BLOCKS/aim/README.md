# AIM

Comprehensive library of atomic MOLT Aim blocks. Aim answers: 'What are we trying to achieve?' Each block describes the desired real-world outcome - what success looks like when this reasoning path completes.

## Family role
AIM answers: 'What are we trying to achieve?' It frames the target outcome or success condition for the reasoning path.

## How to browse
Use `INDEX.md` for category-grouped browsing, or open individual `AIM.###-*.md` files for one block at a time.

## Relationship to other families
- AIM gives the destination
- NEED gives the prerequisite or required condition
- USE gives the practical function or application
- DIRECTIVE tells the system what to do
- INSTRUCTION gives procedural guidance

## Files
- `INDEX.md` - grouped block index
- `AIM-LIBRARY-v1.0.0.md` - human-facing family overview
- `AI/MOLT-BLOCKS/aim.library.v1.0.0.json` - machine-facing source library
