# AIM.005 - Validate assumption

**Type:** AIM
**Category:** ANALYTICAL_OUTCOMES
**Group Description:** Aims oriented toward understanding, classifying, and explaining.
**Status:** active

## Definition
To confirm or refute a stated belief with evidence before proceeding.

## When to use
Aims oriented toward understanding, classifying, and explaining.

## Examples
- checking that a service is actually down
- verifying a requirement is in scope
- testing that a hypothesis holds against data
