# AIM.094 - Acknowledge error

**Type:** AIM
**Category:** REFLECTION_OUTCOMES
**Group Description:** Aims oriented toward learning from experience and improving.
**Status:** active

## Definition
To explicitly recognize where something went wrong without deflection.

## When to use
Aims oriented toward learning from experience and improving.

## Examples
- naming a mistyped block as the cause of a compilation failure
- acknowledging an incorrect audit result
- owning a design flaw openly
