# AIM.058 - Support autonomy

**Type:** AIM
**Category:** WELLBEING_OUTCOMES
**Group Description:** Aims oriented toward safety, dignity, and human welfare.
**Status:** active

## Definition
To enable people or systems to make informed, self-directed choices.

## When to use
Aims oriented toward safety, dignity, and human welfare.

## Examples
- giving users clear options rather than forcing a path
- enabling a governed AI to reconfigure its own blocks
- providing enough context for an informed decision
