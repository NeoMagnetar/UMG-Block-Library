# AIM.032 - Sequence work correctly

**Type:** AIM
**Category:** PLANNING_OUTCOMES
**Group Description:** Aims oriented toward organizing, sequencing, and preparing work.
**Status:** active

## Definition
To order tasks so dependencies are resolved before dependent work begins.

## When to use
Aims oriented toward organizing, sequencing, and preparing work.

## Examples
- ordering MOLT block authoring before compilation
- scheduling infrastructure before app deployment
- putting governance pass before trigger pass
