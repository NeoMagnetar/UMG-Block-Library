# AIM.080 - Prevent duplication of effort

**Type:** AIM
**Category:** COORDINATION_OUTCOMES
**Group Description:** Aims oriented toward aligning people, teams, and systems.
**Status:** active

## Definition
To ensure the same work is not done multiple times unnecessarily.

## When to use
Aims oriented toward aligning people, teams, and systems.

## Examples
- checking the block library before authoring new blocks
- reviewing existing sleeves before building a new one
- aligning teams on who owns which component
