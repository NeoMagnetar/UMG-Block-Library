# AIM.018 - Eliminate uncertainty

**Type:** AIM
**Category:** DECISION_OUTCOMES
**Group Description:** Aims oriented toward making choices and committing to action.
**Status:** active

## Definition
To reduce ambiguity to the point where a sound decision is possible.

## When to use
Aims oriented toward making choices and committing to action.

## Examples
- resolving an open ODR
- clarifying a requirement before build
- confirming a dependency version
