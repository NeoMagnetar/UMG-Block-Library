# AIM.065 - Create optionality

**Type:** AIM
**Category:** STRATEGIC_OUTCOMES
**Group Description:** Aims oriented toward positioning, advantage, and long-term capability.
**Status:** active

## Definition
To preserve the ability to pivot or choose different paths later.

## When to use
Aims oriented toward positioning, advantage, and long-term capability.

## Examples
- keeping ODRs open until empirically resolved
- building a modular sleeve that supports reconfiguration
- avoiding lock-in to a single platform or partner
