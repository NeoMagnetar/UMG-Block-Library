# USE.004 - Data filtering

**Type:** USE
**Category:** DATA_PROCESSING
**Group Description:** Uses related to transforming, moving, or operating on data.
**Status:** active

## Definition
To select relevant subsets from a larger dataset.

## When to use
Uses related to transforming, moving, or operating on data.

## Examples
- querying by date range
- excluding null records
- isolating high-priority items
