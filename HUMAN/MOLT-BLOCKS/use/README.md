# USE

Comprehensive library of atomic MOLT Use blocks. Use answers: 'What is this for?' Each block describes the function or purpose of the Subject - what that thing does or enables.

## Family role
USE answers: 'What is this for?' It frames the practical function, operational purpose, or application of a thing.

## How to browse
Use `INDEX.md` for category-grouped browsing, or open individual `USE.###-*.md` files for one block at a time.

## Relationship to other families
- USE explains application
- AIM explains destination
- NEED explains requirement
- BLUEPRINT explains structure
- INSTRUCTION explains procedure

## Files
- `INDEX.md` - grouped block index
- `USE-LIBRARY-v1.0.0.md` - human-facing family overview
- `AI/MOLT-BLOCKS/use.library.v1.0.0.json` - machine-facing source library
