# Nonprofit Financing Agent

## What this sleeve does
Nonprofit Financing Agent is a finance-domain sleeve for helping nonprofits assess funding readiness, discover grants, build donor pipelines, navigate federal funding, use earned revenue, and manage a diversified funding portfolio.
