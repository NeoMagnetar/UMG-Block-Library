Not every HUMAN sleeve summary is a public runnable machine sleeve. HUMAN coverage may grow faster than machine-facing promotion.

## What these files are
These are human-readable sleeve summaries meant to explain:
- what a sleeve does
- what NeoStacks it contains
- what NeoBlocks are represented or inferred
- what MOLT block families / block names are inside it when that information is available

## What these files are not
These are not automatically machine-facing sleeve JSON payloads.
A HUMAN sleeve entry may be:
- backed by a real machine-facing public JSON sleeve
- backed by a stronger private sleeve-library source and promoted into public HUMAN form
- a lighter placeholder/public helper summary when a full private extraction is not yet promoted

## Current public interpretation levels
Use these informal posture levels when reading the HUMAN sleeve lane:
- **strongly grounded**: backed by rich private sleeve-library materials and/or a real public machine-facing sleeve
- **public summary**: good public HUMAN summary, but not a full machine-facing promotion
- **light placeholder**: useful public category entry, but intentionally conservative because the deeper source extraction is not yet promoted
